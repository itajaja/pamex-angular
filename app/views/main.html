The system is designed to cover basic order management functionality for small to medium sized CNC-machining shops. It is based on a lightweight installation of Simatic IT using only the core components POM (Production Order Manager), MM (Material Manager), BPM (Business Process Modeler) and DIS (Data Integration Service). The system demonstrates integration to Teamcenter via Shop Floor Connect, the Siemens DNC solution, as well as LMS/OEE and tool management components of the Siemens portfolio.
